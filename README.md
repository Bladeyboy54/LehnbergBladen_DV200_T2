## About The Project

### Project Description

The goal of this project was to create a management portal for a medical practice that allowed a receptionist to log in and add, remove or update data relating to patients, doctors, appointments or lower rank nurses.

### Built With

* [PHP](https://reactjs.org/)
* [mySQL](https://www.chartjs.org/)

## Getting Started

The following will allow you to get a copy of the project and have it running on your local machine.

### PRerequisites

Install `xampp`

### How to Install

1. Clone Repository </br>
Run the following in the command-line to clone the project:
   ```sh
   git clone https://github.com/Bladeyboy54/LehnbergBladen_DV200_T2.git
   ```
2. SQL Setup </br>
Import the `clinic.sql` file into phpMyAdmin 

3. Run programme
Run the following in the URL to start the PHP Programme:
   ```sh
   http://localhost/LehnbergBladen_221146_DV200_T2/LehnbergBladen_DV200_T2/index.php
   ```

## Features and Functionality 

- Log into the dashboard
- validate that the eamil and password is correct 
- create a session that allows the receptionist's data to be kept
- display the receptionist's name on the dashboard
- add appointments from the dashboard
- view the upcoming appointments 
- remove appointments accordingly
- navigation allowing the receptionist to switch tabs between appointment, patients and doctors
- add new patients 
- remove patients 
- update patient data
- add new doctors 
- remove doctors 
- update doctor data 
- log off and terminate the session 

### Implementation Process

* Utilizing `php` and `mySQL` to create a dashboard.
* connecting to the `phpMyAdmin` database within the `db.php` file.
* Utilizing CRUD Functionality to Create, Read, Update and Delete data correctly. 

### Challenges

* Correctly launching and creating data in phpMyAdmin
* Getting the data input to be in the correct format to be placed into the `mySQL` database.
* Setting up the CRUD functionality so that it is user-friendly and not confusing
